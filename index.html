Repository Graph AI Coding Agent

Built a repository-level coding workflow for LLM systems using AST-based graph representations of functions, classes, modules, and dependencies. Instead of passing entire codebases into the model, the system retrieves targeted context through graph traversal to support code editing, review, and reasoning tasks while reducing token usage and irrelevant context.
